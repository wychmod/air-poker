# 《噬谎者》空气扑克篇参考素材

下载日期：2026-06-21

本目录用于保存《嘘喰い / 噬谎者》空气扑克篇相关的视觉参考素材，供 Air Poker 网页游戏后续设计阶段做本地学习、风格分析和 moodboard 使用。

## 使用边界

这些图片来自官方或授权公开页面，仍然属于受版权保护的作品素材。

本项目当前定位是非商用、爱好者学习项目。即便如此，也建议把这些文件当作“参考素材”，不要直接当作可发布、可商用、可再分发的生产素材。

适合使用：
- 本地设计参考
- moodboard
- 风格分析
- 构图、光影、压迫感、色调研究
- 辅助推导原创 UI 和原创视觉资产

不建议直接用于：
- 公开网站首屏大图
- 游戏正式背景图
- 宣传海报
- 社交分享图
- 应用商店素材
- 任何未确认授权的公开发布内容

如果后续项目要公开上线，建议使用原创视觉资产：例如原创水下场景、金属数字牌、Air-BIOS 气泡、扑克桌界面和心理战氛围图，而不是直接复制漫画原图。

## 后续设计建议

后续真正做网站或游戏 UI 时，建议从这些素材中提炼“设计语言”，而不是直接复用原图。

可以提炼的方向：
- 水下空间：深色背景、低饱和蓝绿、局部高亮
- 空气资源：气泡、刻度、氧气舱、压力警告
- 数字牌：金属牌、刻印数字、冷光边缘
- 扑克手牌：实体卡牌、可解组合、风险标记
- 心理战：强对比人物剪影、压迫式留白、紧张倒计时
- 灾厄：重叠牌高亮、警戒色、空气损失动画

推荐做法：
1. 用这些素材确定视觉关键词。
2. 画原创 UI 草图。
3. 生成或绘制原创背景和组件资产。
4. 只在设计文档或内部 moodboard 中引用这些参考图。
5. 公开页面使用原创资产，降低版权风险。
